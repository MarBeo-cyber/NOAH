# NOAH v0.3 Architecture

## Runtime

```text
CognitiveNode[]
  -> Consent + Privacy Governor
  -> Rosetta Concept Projection
  -> Collective Cognitive Field
  -> Tension Map
  -> XAAA Wisdom Hint
  -> Bridge Prioritizer
  -> TAAA Prebrief
```

## Core components

| Component | Responsibility |
|---|---|
| CognitiveGeometryEngine | concept-relevant pairwise schema distance |
| CollectiveCognitiveField | centroid, dispersion, tension score |
| PrivacyGovernor | consent, k-anonymity, bounded noise |
| ExperienceExchangeProtocol | XAAA Wisdom Trace ingestion |
| BridgePrioritizer | TAAA-ready bridge queue |
| NOAHCollectiveFieldOrchestrator | end-to-end prebrief |
